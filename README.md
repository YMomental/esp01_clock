# Автономний мікропроцесорний годинник на ESP-01

Проект мережевого годинника з функцією безперебійного живлення (UPS) та візуальною індикацією стану системи. Пристрій автоматично переходить на живлення від акумуляторів при зникненні мережевої напруги, продовжуючи синхронізувати та відображати час.

## Апаратне забезпечення
* **Мікроконтролер**: ESP8266 (модуль ESP-01)
* **Дисплей**: Складова матриця на базі контролерів MAX7219
* **Акумулятори**: 2 шт. LiFePO4 (форм-фактор 32700)
* **Контролер заряду/живлення**: Модуль XL4015 (налаштований як джерело живлення та зарядний пристрій з обмеженням струму)
* **Захист АКБ**: Схема захисту від загальної перенапруги та покомпонентний захист (балансир) кожної банки на базі керованих стабілітронів **TL431**
* **DC-DC перетворювач (5V)**: Модуль типу Mini360 / MP1584 (для живлення дисплеїв)
* **LDO стабілізатор (3.3V)**: Для живлення мікроконтролера ESP-01
* **Плата узгодження**: Мікросхема логіки **К555ЛЛ1** (4 елементи "АБО") для перетворення рівнів керуючих сигналів з 3.3В у 5В.

## Архітектура живлення
1. Основне живлення та зарядка реалізовані на базі модуля **XL4015**, який одночасно працює як системний блок живлення і як CC/CV зарядка для збірки акумуляторів LiFePO4.
2. Батарея захищена від перезаряду загальним модулем контролю, а також індивідуальними схемами на **TL431**, які запобігають перенапрузі на кожному окремому акумуляторі (комірці).
3. При вимкненні мережевого живлення, напруга з батарей подається на понижуючий DC-DC перетворювач, який стабілізує напругу до **5В** для живлення матриць MAX7219.
4. З лінії 5В додатковий лінійний стабілізатор формує **3.3В** для живлення ESP-01.
5. Оскільки мікроконтролер працює з логікою 3.3В, а матриці потребують 5В, сигнали шини даних (DIN, CLK, CS) проходять через логічний вентиль К555ЛЛ1, який працює як Level Shifter.

## Індикація стану
На матриці реалізовано непомітну точкову індикацію для моніторингу статусу пристрою. Індикатори розташовані у крайньому правому стовпчику дисплея:
* **Робота від батареї:** Якщо зникає мережева напруга 220В, загоряється один піксель у **верхньому правому** куті матриці (координата 31, 0).
* **Втрата синхронізації часу:** Якщо годинник втрачає з'єднання з мережею/інтернетом і дані від NTP-сервера не оновлювалися, загоряється один піксель у **нижньому правому** куті матриці (координата 31, 7).
